# Kurasu 6 — Repaso y práctica | これは～のです y patrones de posesión
📅 2026-04-18
Clase de repaso: vocabulario del capítulo 2 y patrones de posesión con の.

## Patrones de posesión
### ¿Esto es de ~? — これは～のですか？
> これは Wanさん の ですか？ — ¿Esto es de Wan?
> いいえ、Wanさん の じゃ ありません。 — No, no es de Wan.

### ¿De quién es este objeto? — この～はだれのですか？
> この かばん は だれ の ですか？ → やまださん の です。 — Es de Yamada.
> この カメラ は だれ の ですか？ → Santosuさん の です。
> この てちょう は だれ の ですか？ → Miraaさん の です。

## 「の」 como pronombre
Cuando el objeto ya es conocido, se omite el sustantivo y の queda como pronombre:
> これ は ハリーポッター の ほん です。 — Esto es el libro de Harry Potter.
> この ほん は わたし の です。 — Este libro es mío (ほん se sobreentiende).
**わたし の です** = es mío.

## Patrones de conversación
| Pregunta | Positiva | Negativa |
|---|---|---|
| これは ～さん の ですか？ | はい、～さん の です。 | いいえ、～さん の じゃ ありません。 |
| この ～ は だれ の ですか？ | ～さん の です。 | いいえ、～さん の じゃありません。～さん の です。 |
