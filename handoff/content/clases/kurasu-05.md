# Kurasu 5 — ¿De quién es? | だれの, なんの y doble pregunta
📅 2026-04-11
Temas: doble pregunta, なんの (qué tipo), だれの (de quién), idiomas (～ご), そうです/ちがいます, pronombres vs adjetivos demostrativos.

## Doble pregunta | Sujeto は A ですか、B ですか？
Dos preguntas sobre el mismo sujeto se unen para no repetirlo:
> それ は シャープペンシル ですか、えんぴつ ですか？ — ¿Eso es un portaminas o un lápiz?
> それ は 「ね」 ですか、「れ」 ですか？ — ¿Eso es "ne" o "re"?

## ¿Qué tipo de...? | なん の ～ ですか？
> それ は なん の かぎ ですか？ — ¿Esa llave es de qué?
> それ は くるま の かぎ です。 — Es la llave del auto.
> それ は なん の ざっし ですか？ → Anime の ざっし です。 — Revista de anime.

## ¿De quién es? | だれ の ～ ですか？
> それ は だれ の かぎ ですか？ → これ は やまださん の かぎ です。 — Es la llave de Yamada.
> それ は だれ の かさ ですか？ → これ は たむらさん の かさ です。 — Es el paraguas de Tamura.

## Idiomas | くに + ご
| País | Idioma | Romaji |
|---|---|---|
| にほん Japón | にほんご | nihongo |
| ちゅうごく China | ちゅうごくご | chuugokugo |
| かんこく Corea | かんこくご | kankokugo |
| アメリカ/イギリス | えいご | eigo (inglés) |
| スペイン | スペインご | supeingo |
| イタリア | イタリアご | itariago |
| フランス | フランスご | furansugo |
| ドイツ | ドイツご | doitsugo |

💡 ご significa "lenguaje/habla". Sufijo sobre el nombre del país; funciona en la mayoría de los casos.

## Confirmar y negar | そうです y ちがいます
- はい、そう です。 (Hai, sou desu.) — Sí, así es.
- いいえ、ちがいます。 (Iie, chigaimasu.) — No, no es así.
> それ は くるま の かぎ ですか？ → いいえ、ちがいます。うち の かぎ です。 — No, es la llave de mi casa.

## Pronombres vs. adjetivos demostrativos
| Tipo | Japonés | Uso | Ejemplo |
|---|---|---|---|
| Pronombre | これ / それ / あれ | Van solos | これ は ほん です |
| Adjetivo | この / その / あの | + sustantivo obligatorio | この ほん は わたし の です |
